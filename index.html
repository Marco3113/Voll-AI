📊 Glossario DataVolley
K1, K2, efficienza, positività…
K1 (side-out) — fase battuta avversaria → cambio palla. Target élite: >60%
K2 (break point) — fase battuta propria → punto diretto. Target: >25%
Efficienza — % punti diretti − % errori diretti (range −100/+100)
Positività — % azioni con esito positivo (non include errori)
A/B/C/D — qualità ricezione: A = alzata perfetta, D = alzata fuori
Jump Serve eff. — target top: 12-18% pt, <8% errori
